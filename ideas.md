# 杨凌锋个人网站设计方案

## 方案一：学术极简主义 (Academic Minimalism)
<response>
<idea>
**Design Movement**: 包豪斯学派 + 当代学术排版
**Core Principles**:
1. 严格的网格系统，黄金比例分割版面
2. 文字即设计，排版层次清晰
3. 留白为主角，内容呼吸感强
4. 功能优先，装饰极少

**Color Philosophy**: 纯白底色 + 深炭黑文字 + 单一强调色（学术蓝 #1a3a5c），传递严谨、专业、可信赖的学术气质

**Layout Paradigm**: 左侧固定导航栏 + 右侧内容区，非对称布局，内容区宽窄交替

**Signature Elements**:
1. 粗细对比强烈的标题线条
2. 数字编号的章节标记
3. 简洁的时间轴

**Interaction Philosophy**: 克制的悬停效果，内容淡入，无多余动效

**Animation**: 页面滚动时内容从左侧滑入，简洁流畅

**Typography System**: Playfair Display（标题）+ Source Serif Pro（正文）
</idea>

</response>

## 方案二：科技感暗黑风 (Tech Dark Mode)
<response>
<idea>
**Design Movement**: 赛博朋克 + 终端美学
**Core Principles**:
1. 深色背景，霓虹色强调
2. 代码/终端风格的视觉语言
3. 几何图形与数据可视化
4. 动态粒子背景

**Color Philosophy**: 深夜蓝黑 (#0a0e1a) + 电光青 (#00d4ff) + 荧光绿 (#39ff14)，展现计算机科学背景的技术气质

**Layout Paradigm**: 全屏分区，每个section占满视口，视差滚动

**Signature Elements**:
1. 打字机效果的文字动画
2. 数据流/矩阵背景
3. 霓虹发光边框

**Interaction Philosophy**: 鼠标跟随光效，悬停时元素发光

**Animation**: 复杂的入场动画，粒子系统，代码雨效果

**Typography System**: JetBrains Mono（标题）+ Fira Code（正文）
</idea>
<probability>0.07</probability>
</response>

## 方案三：温暖现代学者风 (Warm Modern Scholar)
<response>
<idea>
**Design Movement**: 北欧现代主义 + 东方书卷气
**Core Principles**:
1. 温暖米白色基调，营造纸张质感
2. 不对称版面，打破传统简历布局
3. 精致的手工感细节
4. 内容与留白的动态平衡

**Color Philosophy**: 象牙白 (#faf8f3) + 深墨棕 (#2c1810) + 朱砂红 (#c0392b) 作为强调色，传递人文气质与学术热情的平衡

**Layout Paradigm**: 顶部大标题区 + 错落卡片布局，非中心对齐，内容块有机分布

**Signature Elements**:
1. 手绘风格的装饰线条
2. 章节标题旁的几何装饰块
3. 数字统计的动态计数器

**Interaction Philosophy**: 卡片悬停时微微抬起，有纸张翻动的质感

**Animation**: 滚动触发的内容渐入，数字计数动画，平滑过渡

**Typography System**: Noto Serif SC（中文标题）+ Crimson Pro（英文标题）+ Noto Sans SC（正文）
</idea>
<probability>0.09</probability>
</response>

## 选定方案：方案三 - 温暖现代学者风

选择理由：杨凌锋同学兼修金融与计算机，既有理工科的严谨，也有文科的人文气质。温暖现代学者风能够完美平衡这两种特质，不对称的版面布局避免了"AI模板"的刻板感，手工质感的细节体现出个人品味。
